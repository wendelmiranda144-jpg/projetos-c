# Projetos em C

Este repositório contém programas e exercícios desenvolvidos durante meus estudos iniciais na linguagem C, com foco em lógica de programação e fundamentos da linguagem.

## Objetivo
Praticar conceitos básicos de programação em C, conforme aprendido na faculdade e em estudos complementares.

## Conteúdo
- Exercícios de lógica
- Programas simples em C
- Exemplos práticos para fixação da sintaxe

## Tecnologias utilizadas
- Linguagem C
- Compilador GCC (ou compatível)

## Observações
Os códigos aqui presentes têm caráter educacional e representam minha evolução no aprendizado da linguagem C.
